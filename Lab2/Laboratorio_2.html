from sklearn.tree import DecisionTreeClassifier
from sklearn.model_selection import train_test_split

# Droppear categoricas
df_clf = diabetes.drop(columns=['gender', 'smoking_history'])
# Separar atributos y target
X = df_clf.drop(columns=['diabetes'])
y = df_clf['diabetes']
# Separar conjuntos de entrenamiento y test
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=0)
# Entrenar modelo
clf = DecisionTreeClassifier(max_depth=3, random_state=0)
clf.fit(X_train, y_train)
# Obtener predicciones
y_pred = clf.predict(X_test)

  1. Podemos decir que el modelo es bueno segn su curva ROC? Para que podemos usar esta visualizacin?
    2. 























R: La curva ROC muestra un AUC de 0.90, lo que apriori sugiere buen poder discriminativo. Sin embargo, al enfocarse solo en TP y FP no refleja el impacto de los falsos negativos, que en un contexto mdico son crticos y es justo lo que falla en este caso. Por loque exclusivamente mediante la curva ROC no poodemos asegurar de que sea un buen modelo, se debe complementar con otras mtricas comno el recall, o usar una curva del tipo Precision - Recall, vemosla:


















In[21]:






#extra
from sklearn.metrics import precision_recall_curve, average_precision_score

# Obtener probabilidades de la clase positiva
y_scores = clf.predict_proba(X_test)[:, 1]

# Calcular precision y recall para distintos umbrales
precision, recall, thresholds = precision_recall_curve(y_test, y_scores)

# Calcular promedio del rea bajo la curva
ap = average_precision_score(y_test, y_scores)

# Graficar
plt.figure(figsize=(6,5))
plt.plot(recall, precision, color='blue', label=f'Curva PR (AP = {ap:.2f})')
plt.xlabel('Recall')
plt.ylabel('Precision')
plt.title('Curva Precision-Recall')
plt.legend()
plt.grid(True)
plt.show()
















































La curva PR (AP = 0.73) indica buena precisin con bajo recall, pero al aumentar la cobertura de casos positivos la precisin cae rpidamente, reflejando el impacto del desbalance de clases que mencionamos anteriormente.
